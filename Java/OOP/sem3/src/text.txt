/*
На языке программирования Java реализовать классы следующих геометрических фигур:
1.Треугольник
2.Прямоугольник
3.Квадрат
4.Круг
У каждой из этих фигур реализовать методы вычисления периметра и площади этих фигур.
Так же для каждой фигуры реализовать проверку на корректность данных при создании фигуры,
то есть можем ли мы начертить такой треугольник с заданными сторонами или квадрат или прямоугольник.
Так же круг может существовать только с радиусом больше нуля. Использовать эти классы в main и проверить все методы и
конструкторы классов.
 */

 1. Создать абстрактный класс figure.Figure
 1.1 Создать абстрактный метод area
 1.2 Создать абстрактный метод perimeter

 2. Создать класс figure.polygon.triangle.Triangle, наследованный от figure.Figure
 2.1 Создать три поля, отвечающие за стороны
 2.2 Проверять в конструкторе чтобы сумма двух сторон не превышала третюю, для каждой из сторон.
 2.3 Перегрузить метод perimeter: a + b + c
 2.4 Перегрузить метод area: sqrt(p(p-a)(p-b)(p-c)), где р = (a+b+c)/2

 3. Создать класс figure.polygon.rectangle.Rectangle, наследованный от figure.Figure
 3.1 Создать два поля, отвечающие за стороны
 3.2 Перегрузить метод perimeter: 2 * (a + b)
 3.3 Перегрузить метод area: a * b

 4. Создать класс figure.circle.Circle, наследованный от figure.Figure
 3.1 Создать поле, отвечающее за радиус (r)
 3.2 Проверять в конструкторе что r > 0
 3.3 Перегрузить метод perimeter: 2 * pi * r
 3.4 Перегрузить метод area: pi * r * r

5. Создать класс figure.polygon.square.Square, наследованный от figure.polygon.rectangle.Rectangle
5.1 Перегрузить конструктор, который будет принимать одно число (сторону)
